# wiley

## Run Dockerized with nginx and node
```
./devserver.sh
```

## Run in development
```
npm run dev
```

## Build and deploy to firebase hosting
```
npm run deploy
```

### Compiles and hot-reloads for development W/O express
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run Unit tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
